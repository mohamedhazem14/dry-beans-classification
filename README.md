# 🌱 Dry Beans Classification

## 📌 Project Overview
This project focuses on classifying different types of dry beans using machine learning techniques.  
The goal is to predict the bean type based on physical and shape-related features.

The dataset contains 7 different classes of beans:
- HOROZ
- SEKER
- DERMASON
- SIRA
- BARBUNYA
- CALI
- BOMBAY

---

## 📊 Dataset Description
The dataset contains 10834 samples and 17 features + target column.

---

## ⚙️ Workflow
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature scaling using StandardScaler
- Train/Test split (80/20)

---

## 🤖 Models Used
- Extra Trees Classifier (Feature Importance)
- Random Forest Classifier (Final Model)

---

## 📈 Evaluation
- Accuracy Score
- Classification Report
- Confusion Matrix (Normalized & Raw)

---

## 🏆 Result
- Best Model: Random Forest Classifier  
- Accuracy: (91.8%)
